What is MediaMod?

MediaMod is a mod for Minecraft Forge 1.8.9, it will display a music player on your screen with information about the song you are currently playing! The streaming services that are currently supported are:

  • Spotify
  • Google Play Music
  • SoundCloud
  • YouTube
  • YouTube Music
  • Apple Music (web beta)

Try it out!

To download and install MediaMod for Forge 1.8.9, follow these steps:

  1. Download the latest version of MediaMod

  2. Depending on your operating system, the following step will differ

    • Linux:
      Open Terminal and type the following command: cd ~/.minecraft

      Then, type this command: mv /path/to/mod.jar mods/


    • Windows:
      Press the Windows key + R, in the dialog that appears type: %appdata%\.minecraft

      Then, drag the MediaMod jar file into the mods directory


    • macOS / OS X:
      In Finder, from the Go menu, select Go to Folder, then type: ~/Library/Application Support/minecraft, and click Go.

      Then, drag the MediaMod jar file into the mods directory


  3. Open the Minecraft Launcher and start Forge 1.8.9 and enjoy MediaMod!
