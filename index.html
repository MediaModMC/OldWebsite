What is MediaMod?

MediaMod is a mod for Minecraft Forge 1.8.9 & 1.12.2. MediaMod will display a visual representation of the current track on your screen with information about the track. The streaming services that are currently supported are:

  • Spotify
  • SoundCloud
  • YouTube
  • YouTube Music
  • Apple Music

Try it out!

To download and install MediaMod for Forge, follow these steps:

  1. Download the latest version of MediaMod for your Minecraft Version

  2. Depending on your operating system, the following step will differ

    • Linux:
      Open Terminal and type the following command: cd ~/.minecraft

      Then, type this command: mv /path/to/mod.jar mods/


    • Windows:
      Press the Windows key + R, in the dialog that appears type: %appdata%\.minecraft

      Then, drag the MediaMod jar file into the mods directory


    • macOS / OS X:
      In Finder, from the Go menu, select Go to Folder, then type: ~/Library/Application Support/minecraft, and click Go.

      Then, drag the MediaMod jar file into the mods directory


  3. Open the Minecraft Launcher and start Forge, enjoy MediaMod!

Browser Integration

  • If you are on v1.0, download the latest version of the extension from your web store and enjoy!
    Note: you may need to restart your Minecraft Client after installing the extension for it to take effect!

  • If you are on the 2.0 beta, you have to download it from the 2.0.0-beta.1 pre-release page. You can find instructions here.

Updating MediaMod

To update MediaMod, if you are on version 2.0 or higher, you can use the /mmupdate command to automatically update MediaMod. If you are not on version 2.0, go back to the installation instructions
